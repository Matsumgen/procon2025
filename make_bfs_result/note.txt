1. solve1の内容
  上端, 右端を時計回りに揃える。進行方向と垂直に揃える。
  例(上端の場合):
    | | | | - -
    | | | | - -

2. セーブデータの内容
  T : 操作の数
  N : 操作の選択肢の数
  M : 各手順の数
  X : 操作前のx
  Y : 操作前のy
  x : 操作のx
  y : 操作のy
  n : 操作のn

  実際は全て1行
  T
  N0
  M0 x0 y0 z0 x1 y1 z1 x2 y2 z2 ... x(M0 - 1) y(M0 - 1) z(M0 - 1)
  M1 ...
  M2 ...
  ...
  M(N0 - 1) ...
  N1
  ...
  N2
  ...
  ...
  N(T - 1)
  ...
